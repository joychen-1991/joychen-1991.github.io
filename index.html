七牛云qshell操作

  • 下载qshell工具，传到服务器上
  • 添加认证，在七牛云密钥管理中打开
    qshell account AK SK NAME (NAME 自定义即可)
  • 进入.qshell，添加批量上传目录配文件
    vim upload.json rescan_local:扫描新增文件，且上传，默认新增不上传
    overwrite:覆盖文件，如有修改
  • 批量上传命令
    /opt/qshell qupload upload.json
  • 列出bucket文件列表
    /opt/qshell listbucket syhdoctor2020-staging > del.txt
  • 批量删除文件
    /opt/qshell batchdelete syhdoctor2020-staging -i del.txt
  • 更详细的文档访问
    https://developer.qiniu.com/kodo/tools/1302/qshell
  • 添加CDN刷新的目录列表
    vim url.txt
    https://patest.syhdoctor.com/web/syhdoctor
  • 刷新目录
    /opt/qshell cdnrefresh –dirs -i url.txt

Docker离线环境部署以及镜像导入

  • 下载docker源码包
    https://download.docker.com/linux/static/stable/x86_64/
  • 解压
    tar -zxf docker-19.03.8.tgz
  • 将解压文件复制到/usr/bin
    cp docker/* /usr/bin
  • 设置启动服务
    vim /etc/systemd/system/docker.service
  • 增加权限
    chmod +x /etc/systemd/system/docker.service
  • 启动项生效
    systemctl daemon-reload
  • 从其他可以连接网络的服务器上导出镜像
    docker save -o java.tar java
  • 通过工具上传到离线服务器，然后导入
    docker load -i java.tar
  • 运行一个jar包程序实例
    docker run -d –name doctor -v /opt/logs:/logs -v /opt/syh.jar:/opt/syh.jar java java -jar /opt/syh.jar –spring.profiles.active=dev
